# port-town-demo

A minimal static H5 text adventure demo for mobile browsers.

## File structure

```text
port-town-demo/
├─ index.html
├─ style.css
├─ script.js
└─ assets/ (optional)
```

## Run locally

### Option 1
Open `index.html` directly in your browser.

### Option 2 (recommended)

```bash
cd port-town-demo
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Notes

- No build tools
- No framework
- No npm dependencies
- Story flow is defined in `script.js` and can be expanded by adding nodes
